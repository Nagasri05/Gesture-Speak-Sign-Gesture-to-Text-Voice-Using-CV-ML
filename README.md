# 🖐️ Gesture Speak  
### Sign Language to Text & Voice Conversion using CV & ML

## 📌 Overview
Gesture Speak is a system that converts hand gestures (sign language) into readable text and audible speech.  
It aims to help people with speech or hearing impairments communicate easily with others.

---

## 🎯 Problem Statement
Communication is a major challenge for individuals who rely on sign language, especially when interacting with people who do not understand it.  
This project provides a real-time solution to bridge this communication gap.

---

## 💡 Solution
- Capture hand gestures using a camera
- Detect and recognize gestures using Computer Vision
- Convert gestures into text using Machine Learning models
- Convert text into speech using Text-to-Speech technology

---

## 🧠 Technologies Used
- Python  
- OpenCV (Computer Vision)  
- Machine Learning (CNN / Classification Models)  
- NumPy, Pandas  
- Text-to-Speech (TTS)
---

## ⚙️ How It Works
1. Capture real-time video input  
2. Detect hand region and extract features  
3. Classify the gesture using trained model  
4. Convert predicted gesture into text  
5. Generate voice output from text
